# avatar-generator

Live at https://guiomar.xyz/avatar-generator/

Built with [Vue 3](https://vuejs.org/), [Quasar](https://quasar.dev/), and [Vite](https://vitejs.dev/).

## Setup
```
npm install
```

### Development server
```
npm run dev
```

### Build for production
```
npm run build
```

### Preview production build
```
npm run preview
```

### Lint
```
npm run lint
```

### Deploy to GitHub Pages
```
npm run deploy
```
